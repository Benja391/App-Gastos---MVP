# 💸 GASTOS - Controlá tus finanzas de verdad

GASTOS es una aplicación web pensada para ayudarte a entender en qué se te va la plata de forma simple, rápida y sin complicaciones.

## 🚀 ¿Qué hace la app?

- Registrar gastos en segundos
- Ver historial completo de gastos
- Filtrar y organizar información
- Categorizar consumos
- Visualizar tus hábitos financieros
- Mantener control real sobre tu dinero

## 🎯 Objetivo del proyecto

Este proyecto nace con una idea clara:

> Hacer que cualquier persona pueda entender y controlar sus finanzas sin esfuerzo.

Está enfocado especialmente en jóvenes que empiezan a vivir solos o quieren organizar mejor su economía personal.

## 🛠️ Tecnologías utilizadas

- Vue.js (SPA)
- Vite
- Firebase Authentication
- Firestore (base de datos)
- TailwindCSS
- Vercel (deploy)
- Vercel Analytics
